# Android App Analyzer v0.3.0 Release Notes

**Release date:** 2026-03-10

## Summary

This release packages the recent distribution and documentation updates into a new version. It aligns the npm package name with the GitHub repository, adds a Gemini CLI extension manifest, and documents installation flows for both Gemini CLI and Dvcode AI.

## Highlights

### npm package naming alignment

- Updated the published package name to `skill-android-app-analyzer`.
- Kept the npm package identity consistent with the GitHub repository name.
- Added a release notes document to make release content easier to track.

### Gemini CLI extension support

- Added a root `gemini-extension.json` manifest.
- Made the repository installable as a Gemini CLI extension directly from GitHub or a local path.
- Added validation instructions for `gemini extensions validate .`.

### Expanded installation documentation

- Documented the Gemini CLI installation flow in both English and Chinese README files.
- Added Dvcode AI marketplace installation instructions in both README files.
- Kept the repository documentation aligned across Codex, Claude Code, Gemini CLI, and Dvcode AI usage paths.
